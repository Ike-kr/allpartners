## 1. 서비스 개요

### 서비스명
**올파트너스 (AllPartners)**

### 한줄 소개
국내 모든 어필리에이트(제휴 마케팅) 프로그램을 한눈에 비교하고, 나에게 맞는 프로그램을 찾을 수 있는 비교 플랫폼이다.

### 서비스 설명
블로거, 유튜버, 인스타그래머 등 콘텐츠 크리에이터가 수익화를 위해 어필리에이트 프로그램에 가입하려 할 때, 각 프로그램의 수수료율, 정산 조건, 참여 채널 등을 개별 사이트에서 일일이 확인해야 하는 불편함이 있다. 올파트너스는 국내 주요 어필리에이트 프로그램의 핵심 정보를 한곳에 모아 비교할 수 있도록 하여, 크리에이터가 최적의 프로그램을 빠르게 선택할 수 있게 돕는다.

### 타겟 사용자

| 구분 | 세부 설명 |
|------|-----------|
| **1차 타겟** | 블로그/유튜브를 시작한 지 1년 이내의 초보 크리에이터. 제휴 마케팅에 관심은 있지만 어디서 시작해야 할지 모르는 사람 |
| **2차 타겟** | 이미 1~2개 어필리에이트 프로그램에 참여 중이며, 추가 프로그램을 탐색하는 중급 크리에이터 |
| **3차 타겟** | 제휴 마케팅 교육 콘텐츠를 만드는 마케터, 에이전시 |

### 해결하려는 문제

1. **정보 분산**: 각 어필리에이트 프로그램의 조건이 각각의 공식 사이트에 흩어져 있어, 비교하려면 최소 10개 이상의 탭을 열어야 한다.
2. **정보 부재**: 수수료율, 정산 주기, 최소 출금액 등 핵심 정보가 체계적으로 정리된 곳이 없다. 산발적인 블로그 포스트에만 의존해야 한다.
3. **비교 불가**: 프로그램마다 표기 방식이 달라(CPS/CPA/CPC 혼용, 수수료율 표기 불일치 등) 직접 비교가 어렵다.
4. **최신성 부족**: 기존 블로그 글은 작성 시점의 정보가 고정되어, 수수료 변경이나 프로그램 신설/종료 정보가 반영되지 않는다.

## 4. 페이지 구성

### 4.1 메인 페이지 (/)

**목적**: 전체 어필리에이트 프로그램을 한눈에 탐색

- **헤더**: 서비스 소개 한 줄 + 검색바
- **필터 영역**:
  - 수수료 유형 필터 (CPS / CPA / CPC / 전체)
  - 카테고리 필터 (종합 / 패션 / 뷰티 / 여행 / 인테리어 / 기타)
  - 참여 채널 필터 (블로그 / 유튜브 / 인스타그램 / 전체)
  - 가입 난이도 필터 (쉬움 / 보통 / 어려움)
- **정렬 옵션**: 수수료율 높은순 / 이름순 / 최근 업데이트순
- **프로그램 카드 리스트**: 카드형 레이아웃으로 프로그램 요약 정보 표시
  - 프로그램명 + 운영사
  - 수수료 유형 뱃지
  - 수수료율 하이라이트
  - 참여 가능 채널 아이콘
  - 카테고리 태그
  - "자세히 보기" / "공식 사이트" 버튼

### 4.2 상세 페이지 (/programs/[id])

**목적**: 개별 프로그램의 상세 정보 확인

- **프로그램 기본 정보**: 이름, 운영사, 로고, 한줄 소개
- **핵심 지표 카드**: 수수료율, 정산 주기, 최소 출금액, 가입 난이도
- **상세 설명**: 프로그램 특징, 주의사항
- **가입 조건**: 필요 요건 상세 안내
- **참여 가능 채널**: 각 채널별 참여 방법
- **공식 사이트 바로가기 CTA 버튼**
- **(향후) 사용자 리뷰/팁 섹션**

### 4.3 비교 페이지 (/compare) — MVP 이후

**목적**: 선택한 2~3개 프로그램을 나란히 비교

- 프로그램 선택 UI (최대 3개)
- 테이블 형태로 모든 비교 항목 나란히 표시
- 차이점 하이라이트 기능

### 4.4 기타 페이지

- **소개 페이지 (/about)**: 올파트너스 서비스 소개, 데이터 업데이트 주기 안내
- **제보/문의 (/contact)**: 정보 오류 제보, 프로그램 추가 요청

### 5.2 필드 타입 정의

```typescript
interface AffiliateProgram {
  id: string;                          // URL 슬러그 (예: "coupang-partners")
  name: string;                        // 프로그램명
  company: string;                     // 운영사
  logo: string;                        // 로고 이미지 경로
  description: string;                 // 한줄 소개

  type: CommissionType[];              // 수수료 유형
  commissionRate: {
    min: string;                       // 최소 수수료율
    max: string;                       // 최대 수수료율
    description: string;               // 수수료 상세 설명
  };
  indirectCommission?: {               // 간접 기여 수수료 (선택)
    available: boolean;
    rate: string;
    description: string;
  };

  settlementCycle: string;             // 정산 주기
  minimumPayout: number | null;        // 최소 출금액 (원), null이면 미확인
  minimumPayoutDescription: string;    // 출금액 설명

  signupDifficulty: "easy" | "medium" | "hard";
  signupRequirements: string[];        // 가입 조건 목록

  channels: Channel[];                 // 참여 가능 채널
  categories: Category[];              // 카테고리

  cookieDuration: string | null;       // 쿠키 유효 기간
  bonusProgram: string | null;         // 보너스 프로그램 설명

  officialUrl: string;                 // 공식 사이트 URL
  warnings: string[];                  // 경고/주의사항

  lastUpdated: string;                 // 최종 업데이트일 (YYYY-MM-DD)
  status: "active" | "inactive" | "warning";  // 운영 상태
}

type CommissionType = "CPS" | "CPA" | "CPC" | "CPM";

type Channel =
  | "blog"
  | "youtube"
  | "instagram"
  | "website"
  | "cafe"
  | "threads"
  | "facebook"
  | "app";

type Category =
  | "종합"
  | "패션"
  | "뷰티"
  | "여행"
  | "인테리어"
  | "식품"
  | "디스플레이광고"
  | "기타";
```

## 6. 기술 스택

| 구분 | 기술 | 선택 이유 |
|------|------|-----------|
| **프레임워크** | Next.js 16 (App Router) | SSG(정적 생성)로 빠른 로딩, SEO 최적화 내장, ISR로 점진적 업데이트 가능 |

| **데이터 저장** | JSON 파일 (정적) | DB 불필요. 데이터 변경 빈도 낮음 (월 1회 수준). Git으로 이력 관리 가능 |
| **배포** | Vercel | Next.js 최적화 호스팅, 무료 플랜 제공, 자동 배포 |
| **SEO** | Next.js Metadata API | 정적 사이트에 적합, 구조화된 데이터 (JSON-LD) 지원 |
| **분석** | Google Analytics 4 (무료) 또는 Vercel Analytics | 방문자 추적, 인기 프로그램 파악 |
| **아이콘** | Lucide React | 경량, 트리셰이킹 지원 |

## 10. 경쟁 분석

### 현재 상황

국내에 어필리에이트 프로그램만을 전문적으로 비교하는 사이트는 존재하지 않는다. 현재 크리에이터가 정보를 얻는 경로는 다음과 같다.

| 경로 | 한계 |
|------|------|
| 개인 블로그 포스트 | 작성 시점 이후 업데이트 안 됨. 특정 프로그램만 다룸 |
| 유튜브 영상 | 영상 형태라 빠른 비교가 어려움 |
| 각 프로그램 공식 사이트 | 자사 정보만 제공, 비교 불가 |
| 네이버 카페/커뮤니티 | 파편적이고 검증 안 됨 |

### 올파트너스의 차별점

1. **원스톱 비교**: 하나의 사이트에서 모든 프로그램 비교 가능
2. **구조화된 데이터**: 동일한 형식으로 정리되어 직접 비교가 쉬움
3. **최신성 유지**: 정기적 데이터 업데이트 (Git 기반 관리)
4. **신뢰성**: 운영 이슈가 있는 프로그램(예: 미정산 사태)에 대한 경고 표시
5. **SEO 최적화**: "어필리에이트 비교", "제휴 마케팅 프로그램 추천" 등 키워드 선점 가능
