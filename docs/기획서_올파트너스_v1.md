# 올파트너스(AllPartners) — 한국 제휴 마케팅 프로그램 비교 플랫폼 기획서 v1

## 1. 서비스 개요

**한줄 요약:** 한국의 모든 제휴 마케팅(파트너스/어필리에이트) 프로그램을 한눈에 비교하고 탐색할 수 있는 정보 플랫폼

**3줄 설명:**
- 쿠팡 파트너스, 11번가, 알리익스프레스 등 다양한 파트너스 프로그램이 존재하지만, 각각의 조건(수수료율, 승인 난이도, 쿠키 기간 등)을 한눈에 비교할 수 있는 곳이 없다.
- 올파트너스는 한국에서 운영 중인 제휴 마케팅 프로그램을 한 곳에 모아, 블로거·인플루언서·부업러가 자신에게 맞는 프로그램을 쉽게 찾을 수 있게 해준다.
- 파트너스 프로그램은 앞으로 더 늘어날 전망이며, 올파트너스는 이 성장에 맞춰 지속적으로 DB를 확장하는 구조로 설계한다.

---

## 2. 문제 정의

### 타겟이 겪는 구체적 문제

**블로거/인플루언서의 문제:**
- 제휴 마케팅을 시작하고 싶은데, **어떤 프로그램이 있는지** 모른다
- 쿠팡 파트너스는 유명하지만, 다른 옵션(11번가, 알리, 테무 등)과 **수수료를 비교하기 어렵다**
- 각 프로그램의 가입 조건, 수수료율, 쿠키 기간, 정산 방식 등을 알려면 **일일이 사이트를 돌아다녀야 한다**
- 카테고리별(뷰티, 전자제품, 패션 등)로 **어디가 유리한지** 정보가 분산되어 있다

**부업/사이드 프로젝트 희망자의 문제:**
- "제휴 마케팅으로 부수입을 올리고 싶다"는 사람이 늘고 있지만, **진입 장벽이 정보 부족**
- 블로그 글마다 "쿠팡 파트너스 후기"만 넘쳐나고, **체계적인 비교 정보가 없다**

## 3. 타겟 사용자

| 구분 | 메인 타겟 | 서브 타겟 |
|------|-----------|-----------|
| 역할 | 제휴 마케팅 블로거/인플루언서 | 부업 관심자, 사이드 프로젝트 희망자 |
| 연령 | 20~40대 | 20~30대 |
| 행동 특성 | 이미 1~2개 파트너스에 가입해서 활동 중, 더 좋은 조건을 찾고 싶음 | 아직 시작 전, 어디서부터 해야 할지 모름 |
| 핵심 니즈 | 프로그램 간 수수료/조건 비교, 카테고리별 최적 선택 | "나한테 맞는 첫 파트너스 프로그램" 추천 |

## 4. 핵심 기능 (최대 3개)

### 기능 1: 파트너스 프로그램 DB + 비교표
- 한국에서 운영 중인 제휴 마케팅 프로그램을 한 곳에 수집
- 각 프로그램별 핵심 정보 정리:
  - 프로그램명 / 운영사
  - 수수료율 (카테고리별)
  - 쿠키 유지 기간
  - 정산 주기 / 최소 정산 금액
  - 가입 조건 / 승인 난이도
  - 제공 도구 (링크 생성기, 배너 등)
  - 지원 채널 (블로그, SNS, 유튜브 등)
- 필터/정렬로 조건에 맞는 프로그램 빠르게 탐색

### 기능 2: 카테고리별 추천
- "뷰티 제품을 주로 리뷰하는데, 어디가 수수료 높아?" 같은 질문에 답하는 카테고리별 추천
- 카테고리: 뷰티, 전자제품, 패션, 식품, 도서, 여행, 생활용품 등
- 각 카테고리에서 가장 유리한 프로그램 하이라이트

### 기능 3: 초보자 가이드
- "제휴 마케팅이 뭐야?"부터 시작하는 입문 가이드
- 각 프로그램별 가입 방법 간단 안내
- FAQ: "수익은 어떻게 나?" "세금은?" "블로그가 꼭 있어야 해?" 등

## 5. 사용자 플로우

### 메인 플로우
```
1. 랜딩페이지 접속 → "한국 파트너스 프로그램 한눈에 비교"
2. 전체 프로그램 목록 확인 (카드형 or 테이블형)
3. 필터 적용: 카테고리 / 수수료율 / 승인 난이도 등
4. 관심 프로그램 클릭 → 상세 정보 확인
5. "가입하러 가기" → 해당 프로그램 공식 페이지로 이동
```

### 초보자 플로우
```
1. "어디서 시작해야 할지 모르겠어요" 클릭
2. 간단 질문: "주로 어떤 콘텐츠를 만드세요?" (블로그/유튜브/SNS)
3. 질문: "주로 다루는 카테고리는?" (뷰티/전자제품/패션 등)
4. 맞춤 추천: "이 프로그램부터 시작해보세요!"
```

## 6. 차별화 포인트

### 기존에 없는 이유
- 각 파트너스 프로그램 공식 사이트에서는 **자기 프로그램 정보만** 제공
- 블로그 후기는 **개인 경험 1~2개** 수준, 체계적 비교가 아님
- 한국에 "제휴 마케팅 프로그램 비교 사이트"는 **현재 존재하지 않음**

### 차별점
| 기존 정보원 | 올파트너스 |
|-------------|-----------|
| 각 프로그램 공식 사이트 (분산) | 한 곳에 모아서 비교 |
| 블로그 후기 (주관적, 1~2개) | 객관적 데이터 기반 비교표 |
| 쿠팡 파트너스만 유명 | 숨겨진 프로그램까지 발굴 |
| 정보 업데이트 느림 | 지속적 DB 확장 구조 |

## 7. MVP 범위

### ✅ MVP에 포함
- 랜딩페이지 (서비스 소개)
- 파트너스 프로그램 DB (최소 10개 이상)
- 비교표 (테이블형, 필터/정렬 가능)
- 각 프로그램 상세 페이지
- 초보자 가이드 (기본)

### ❌ MVP에서 제외 (추후 확장)
- 사용자 리뷰/평점 시스템
- 수익 계산기 (예상 수익 시뮬레이션)
- 맞춤 추천 알고리즘
- 뉴스레터/알림 기능 (새 프로그램 추가 시)
- 커뮤니티 기능
- 회원가입/로그인
